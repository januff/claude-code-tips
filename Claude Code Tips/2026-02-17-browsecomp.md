## Summary

Claude's web search capabilities are now enhanced with code execution for pre-filtering web results. This upgrade allows Claude to refine search results before they are added to the context window, significantly improving accuracy and reducing token usage. Specifically, Sonnet 4.6 demonstrated a 13% accuracy improvement on the BrowseComp task with a 32% reduction in input tokens when this filtering was enabled.
