## Summary

The `ultraplan` command in Claude Code generates an editable implementation plan in a web interface, streamlining complex code refactoring tasks. By using `ultraplan migrate auth to OAuth`, for example, Claude will build a plan you can review and approve before choosing an implementation method from your terminal. This enables a more controlled and understandable refactoring process.
