## Summary

Anthropic is introducing "Auto Mode" for Claude Code, a feature designed to streamline development workflows by automatically handling permission prompts, scheduled for research preview release around March 12, 2026. This eliminates the need for constant manual approval or the riskier `--dangerously-skip-permissions` flag. To enable, use `claude --enable-auto-mode`, keeping in mind the increased token usage and processing time due to security checks and the recommendation to use it in isolated environments like sandboxes for now; team restrictions via MDM tools are also possible.
