Claude Code now supports local plugins on the desktop application, enabling users to extend functionality with custom slash commands, skills, and MCP servers. These plugins can be installed from a marketplace and will automatically synchronize between the desktop app and the CLI, streamlining workflows across different environments. This allows for a more personalized and efficient coding experience.
