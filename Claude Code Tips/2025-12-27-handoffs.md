## Summary

This tip highlights a workflow for managing long Claude Code projects using 'handoffs.' At the end of each coding session, prompt Claude to generate a markdown summary of progress, decisions, open questions, and next steps. This technique improves continuity by providing a concise, easily digestible update for picking up where you left off.
