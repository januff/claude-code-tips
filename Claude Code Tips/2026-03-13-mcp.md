## Summary

This tip showcases a workflow for automating browser interactions using a CLI agent with Chrome CDP (Chrome DevTools Protocol). By enabling a toggle in Chrome 146, you can expose your browsing session for programmatic control. The example demonstrates how to use a script to automatically decline LinkedIn invitations by targeting and clicking the 'Ignore' button, using JavaScript injected via the `cdp.mjs eval` command.
