> PSA: Claude Code automatically DELETES sessions that have been inactive for more than 30 days.
> 
> Disable this by setting "cleanupPeriodDays": 99999 (or some other large number) in ~/.claude/settings.json. Do not ever attempt to disable it by setting that to 0, lmao.
>

## Summary

The Claude Code tip highlights how to prevent automatic deletion of inactive sessions, which are deleted after 30 days by default. You can disable this behavior by modifying the `cleanupPeriodDays` setting in the `~/.claude/settings.json` file to a large number like 99999. Avoid setting this value to 0, as that could lead to unintended consequences.
