## Summary

This Claude Code tip introduces a "theorist" layer to improve long-running agent sessions. Instead of relying solely on static plans and shallow session history, the theorist continuously updates a mental model of the root cause and the current "theory of victory." This helps prevent agents from getting stuck in repetitive loops by providing a deeper understanding of the problem.
