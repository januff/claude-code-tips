## Summary

This tip shows how to create a command-line alias in your `.zshrc` file for the `claude` command, specifically creating a shortcut `ccc` that bypasses permission checks. This allows for quicker execution of the `claude` command by defining `alias ccc="claude --dangerously-skip-permissions"`. Use this technique to streamline interactions with programs, but be mindful of security implications when skipping permissions.
