## Summary

This tip shows how to create a convenient command-line alias, specifically for the 'claude' command. By adding `alias ccc="claude --dangerously-skip-permissions"` to your `.zshrc` file, you can create the shorter `ccc` alias for running claude with the `--dangerously-skip-permissions` flag. This allows for quicker and easier execution of the command, although users should understand the implications of skipping permissions.
