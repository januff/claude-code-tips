## Summary

This tip highlights a crucial organizational technique for building robust agent-based systems, specifically focusing on skill management. The key is to ensure your agent has easy access to documentation by either copying or, preferably, symlinking your skills documentation (xcdocs) to both the `.agents/` and `.claude/` directories, as doing this implies the user understands how to give the agent access to the knowledge. This approach, involving the `copy` or `symlink` command, streamlines the agent's ability to leverage relevant information.
