## Summary

This tip improves interface development workflows using Claude Code by centralizing visual references. The key insight is to create a dedicated folder containing screenshots and other UI references and provide this folder as a default reference to Claude Code. This avoids scattered files and ensures Claude has consistent, accessible context for generating accurate code.
